Получить JAR можно двумя способами:
1)Переходим в папку с проектом и выполняем команду mvn install. После этого JAR файл проекта будет скомпилирован и положен в местный репозиторий.(может не работать)
2)Открываем файл проекта через Intellij IDEA со встроенным плагином Maven. Выполняем фазу install.
Затем переходим в каталог с JAR файлом и выполняем команду java -jar ParagonCase-x.x.jar C:\File\Path\text.txt
